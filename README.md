# Wumpus World Simulator (Streamlit)

Interactive Wumpus World environment + visualization (Russell & Norvig style).

## Features

- WumpusWorld environment with percepts: stench, breeze, glitter, bump, scream
- Emoji-based grid UI (debug view vs hidden view)
- Step / Play / Pause controls
- Full telemetry log + charts + CSV export
- Baseline NaiveAgent (random actions using percept input)

## How to run

1. Create/activate a virtual environment (recommended)
2. Install dependencies:

```bash
pip install -r requirements.txt
```
